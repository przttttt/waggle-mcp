# MemBench (ACL 2025)

MemBench evaluates factual and reflective memory across participation and observation scenarios.

## Download Instructions
Download the dataset from [import-myself/Membench](https://github.com/import-myself/Membench).

## Reproduction Commands
```bash
.venv/bin/python scripts/benchmark_membench.py path/to/membench.json --mode graph --output benchmarks/membench/results_graph_raw.json
```
